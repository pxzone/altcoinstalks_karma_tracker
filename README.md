
# [Userscript] Sent Karma Log Tracker

This userscript work just like how the karma link (positive/negative) works in the forum. And this will show where and who you sent karma in he past 120 days.
Just like how the karma works, this userscript will
- Save Karma from karma link (positive/negative)
- Won't save and only update karma type within 10 hours

## How to install
- Install Tampermonkey either for desktop (most browsers) and mobile (Edge or any browsers supporting extensions).
- Go to your Altcoinstalks Profile Summary page (To get your UID on installation process)
- Go Tampermonkey and Create new script..
- Copy and paste the script from [Github](https://github.com/pxzone/altcoinstalks_karma_tracker/raw/refs/heads/main/karma_tracker.user.js )
- Then save

The only downside is, you need to stay to the page after clicking the karma link, for it to send the payload to the server for saving, so at least waiting 5-10 seconds after clicking is enough.
